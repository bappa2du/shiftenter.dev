# GraphQL

### Introduction

{% hint style="info" %}
**Graphql** support in Magento from version **2.3**
{% endhint %}

{% hint style="warning" %}
The default endpoint is **/graphql** means **https://website.com/graphql**
{% endhint %}

#### Sample Query:

```graphql
{
  storeConfig {
    id
    code
    website_id
    locale
    base_currency_code
    default_display_currency_code
    timezone
    weight_unit
    base_url
    base_link_url
    base_static_url
    base_media_url
    secure_base_url
    secure_base_link_url
    secure_base_static_url
    secure_base_media_url
    store_name
  }
}
```
